# replica

A configurable repl (read-eval-print-loop) capable of running multiple compiled languages (C, C++, D and more to come)

## How to use ?

```sh
matteov> ./repl --env cxx --config configs/
> std::cout << "hello world";
hello world
> exit
matteov>
```
